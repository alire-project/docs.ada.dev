lace

0.1.0

Contains a set of low level re-usable Ada components.

Contains: - lace.Events : Provides a 'subject/oberver' 'event/response' facility. - lace.Any : Provides an interface to allow heterogenous containers. - lace.fast_Pool : Provides a generic which allows fast allocation/deallocation. - lace.Text : Provides a DSA friendly set of text operations.
