printer_toolkit

0.3.0

Printer toolkit to write reports

Printer toolkit provides support to write reports on console or to produce SVG files with almost the same Ada code. It can display simple bar charts as well as images on the console. The core library is provided by `printer_toolkit` and the SVG support implemented by the `printer_toolkit_svg`. Run one of the following Alire commands or both: ``` alr with printer_toolkit alr with printer_toolkit_svg ```
