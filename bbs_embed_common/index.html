bbs_embed_common

0.2.0

Abstract hardware drivers and drivers for some i2c bus devices.

This crate contains base classes for some hardware devices such as Analog Inputs, GPIO pins, I2C bus, and SPI bus. It also contains drivers for devices that use, for example, an I2C bus. By itself, this crate isn't much use. It will need to be used with the bbs_embed_linux crate which contains hardware drivers for the Raspberry Pi and BeagleBone Black.
